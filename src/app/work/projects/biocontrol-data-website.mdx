## Vue d'ensemble

Développement d'un site web technique permettant de comparer des données de biocontrôle. Ce projet combine développement web et visualisation de données pour créer un outil d'analyse interactif.

## Fonctionnalités clés

- **Comparaison de données**: Système de comparaison de données de biocontrôle.
- **Visualisations interactives**: Graphiques et tableaux pour analyser les données.
- **Interface technique**: Design adapté à un usage professionnel et scientifique.
- **Performance**: Traitement efficace des données pour une expérience fluide.

## Technologies utilisées

- **HTML/CSS/JavaScript**: Pour l'interface et les interactions.
- **Visualisation de données**: Bibliothèques pour créer des graphiques interactifs.
- **Traitement de données**: Manipulation et analyse de données de biocontrôle.

## Lien du projet

[Voir le site en ligne](https://my-bio-ivory.vercel.app/html/index.html)

## Défis et apprentissages

Ce projet m'a permis de développer mes compétences en visualisation de données et en traitement de données complexes. Le principal défi était de rendre les données scientifiques accessibles et compréhensibles à travers une interface intuitive.

## Résultat

Un outil web fonctionnel permettant de comparer et d'analyser des données de biocontrôle de manière interactive et visuelle, déployé sur Vercel.
